# 🚀 OpenCV Haar Cascade Detection Suite

A collection of Computer Vision projects developed using **Python** and **OpenCV**. This repository demonstrates how Haar Cascade classifiers can be used to detect faces, eyes, cars, pedestrians, and perform image sketch generation.

---

## 📌 Features

- 😀 Face Detection from Image
- 👥 Multiple Face & Eye Detection using Webcam
- 🚗 Car Detection from Video
- 🚶 Pedestrian Detection from Video
- 🎨 Sketch Generation from Image
- XML Haar Cascade Classifiers Included

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/your-username/opencv-haarcascade-detection-suite.git
```

Move into the project folder

```bash
cd opencv-haarcascade-detection-suite
```

Install dependencies

```bash
pip install -r requirements.txt
```

### Face Detection
